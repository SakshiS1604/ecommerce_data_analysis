# E-Commerce Data Analysis Project

## Project Overview
This project analyzes Amazon product data to uncover insights about pricing, discounts, customer ratings, and product performance.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objectives
- Perform exploratory data analysis (EDA)
- Identify product pricing trends
- Analyze customer ratings and reviews
- Discover high performing products

## Key Insights
- Electronics category has the highest number of products
- Most products are priced between ₹500–₹2000
- Higher discounts tend to attract more reviews
- Premium products receive fewer reviews

## Visualizations
- Price Distribution
- Rating Distribution
- Discount Analysis
- Correlation Heatmap
- Category Analysis

## Project Workflow
1. Data Cleaning
2. Feature Engineering
3. Exploratory Data Analysis
4. Data Visualization
5. Business Insights
